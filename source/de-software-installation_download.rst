===========================================
Drone Engage RPI-WZero2 / RPI-4 / RPI-5 Ready Image
===========================================


DroneEngage has a ready RPI-Image that you can dowload it directly from here:

Download Image from `Here <https://cloud.ardupilot.org/downloads/RPI_Full_Images/>`_ .


.. youtube::  -mV9EARM0_0

|


Once you download the image you need to flash it on SD-CARD 16GB or more.

You need to use app like `Raspberry Pi Imager <https://www.raspberrypi.com/software/>`_ or `balenaEtcher <https://www.balena.io/etcher/>`_ to flash the image on the SD-CARD.

It is important to remember not to apply settings or network configuration from RPI-Imager while flashing the image.

Once you run this image open your laptop and search for a WiFi with the name **DE_ADMIN** and password **droneengage**.

Once you connect to this AP open your browser and go to `https://192.169.9.1:9090 <https://192.169.9.1:9090>`_ 


username/password:  **pi/raspberry**





Now you can use :ref:`de-rpi-image-tools` for easier administration.
